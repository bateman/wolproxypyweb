wolproxypy

wolproxypy is a all-in-one solution for wake on lan. It is made of three main components:

Core

Available through PyPI, wolproxypy core is a python library that provides a simple commandline interface to wake hosts on a lan.

API

Developed using FastAPI, the wolproxypy api expose the core functionality as a RESTful API. The API can be easily deployed via Docker.

Wep app

wolproxypy web is application that provides a simple user interface to manage and wake up hosts from your browser. The app is responsive as it is built using Flask and Bootstrap.

Feel free to contribute to this project on GitHub.
